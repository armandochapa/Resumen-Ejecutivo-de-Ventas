# Resumen-Ejecutivo-de-Ventas

Generación de resumen ejecutivo para decidir determinados ajustes de presupuesto e inventario.

El reto consiste en limpiar, organizar y analizar los datos de ventas semanales de 2012, con el fin de responder a preguntas clave de negocio mediante KPIs bien definidos y visualizaciones claras.

Este proyecto se realiza:

- Limpieza y preparación de datos.
- Creación de métricas de negocio (KPIs).
- Construcción de dashboards interactivos.
- Comunicación ejecutiva con el método C-F-I (Context, Finding, Implication).
